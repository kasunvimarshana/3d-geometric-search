# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-01-XX

### Added

- Initial release of 3D Model Viewer
- Core domain layer with entities (Model, Section, ViewState, Selection)
- 3D rendering engine with Three.js integration
- State management system with observable store
- Event orchestration layer with validation
- UI components (HierarchyTree, PropertiesPanel, StatisticsPanel)
- Multi-format loader support (glTF/GLB, OBJ/MTL, STL)
- Interactive viewport with mouse controls
- Section selection and highlighting
- Camera controls (zoom, pan, rotate, fit, reset)
- Hierarchical model navigation
- Properties panel with section details
- Statistics panel with model metrics
- Wireframe mode toggle
- Grid and axes helpers
- Fullscreen mode
- Drag-and-drop file loading
- Clean, professional UI design
- Comprehensive documentation
- ESLint and Prettier configuration
- Vite build configuration

### Planned

- STEP format support with OpenCascade.js
- Measurement tools
- Section plane cutting
- Annotations
- Animation playback
- Comparison mode
- Explode/assemble animation
- Geometric search
- Collaborative features
