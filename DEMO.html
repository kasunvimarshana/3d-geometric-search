üíª Code Examples

Check Performance Stats:

// Run in browser console getPerformanceStats() // Returns: { sections: { loaded: 1, visible: 1, registered: 3 }, cache: { size: 2, hits: 5, misses: 2 }, components: { geometryAnalyzer: true, exportManager: false } }

Monitor Section Lifecycle:

// Listen to section events document.addEventListener('sectionShown', (e) => { console.log(`Section shown: ${e.detail.sectionId}`); }); document.addEventListener('sectionHidden', (e) => { console.log(`Section hidden: ${e.detail.sectionId}`); });

Programmatic Section Control:

// Toggle a section app.sectionManager.toggleSection('advanced-controls'); // Show a section app.sectionManager.showSection('library-section'); // Hide a section app.sectionManager.hideSection('results-section'); // Check if loaded app.sectionManager.isLoaded('advanced-controls'); // true/false

Cache Management:

// View cached analyses app.analysisCache // Clear cache app.analysisCache.clear() // Get specific cached result app.analysisCache.get('model_name')
